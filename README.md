# 📦 CmdForge 

사용자가 직접 커맨드를 저장하고 실행할 수 있는 Python 기반 커맨드 관리 시스템.

---

## 🔧 기능

* 커맨드 저장 (`set`)
* 커맨드 실행 (`run`)
* 커맨드 삭제 (`del`)
* 전체 삭제 (`delall`)
* JSON 파일을 이용한 데이터 저장

---

## 📁 파일 구조

```
project/
│── main.py
│── data.json (자동 생성)
│── Code.py (사용자 정의 함수 모음)
```

---

## ⚙️ 동작 방식

* 커맨드와 코드를 `data.json`에 저장
* `run` 명령어로 저장된 코드를 실행 (`exec` 사용)
* 존재하지 않는 파일이면 자동 생성

---

## 🚀 사용법

### 1️⃣ 커맨드 저장

```
set.커맨드이름:실행코드
```

예시:

```
set.hello:print("Hello World")
```

---

### 2️⃣ 커맨드 실행

```
run.커맨드이름
```

예시:

```
run.hello
```

출력:

```
hello 실행
Hello World
```

---

### 3️⃣ 커맨드 삭제

```
del.커맨드이름
```

---

### 4️⃣ 전체 삭제

```
delall
```

---

### 5️⃣ 프로그램 종료

```
end
```

---

## ⚠️ 주의사항

### ❗ exec 사용

이 프로그램은 Python의 `exec()`를 사용함

→ 즉, 저장된 코드는 그대로 실행됨
→ 악성 코드 실행 가능성 있음

👉 개인용 / 테스트용으로만 사용하는 것을 권장

---

### ❗ JSON 구조

`data.json` 예시:

```json
[
    {
        "cmd": "hello",
        "code": "print('Hello World')"
    }
]
```

---

## 🧠 내부 로직 요약

* `setcmd()` → 입력 파싱 후 JSON에 추가
* `runcmd()` → cmd 매칭 후 exec 실행
* `delcmd()` → 특정 cmd 제거
* `delallcmd()` → 전체 초기화
* `read_data_from_file()` → JSON 읽기
